StageRehearsal 专业文档

这里整理了公网演示、前端操作、Unreal 实时联动和数据导出的最短路径。公开仓库只包含静态网站和可再分发测试资产。

打开交互 Demo

快速开始

本地预览

http://localhost:8000/demo/

公网访问

https://endtofx.github.io/StageRehearsal/demo/

前端 Demo 使用

页面默认中文，可切换英文。左侧输入创意描述，点击“一键生成预演”即可同时生成动作、运镜和灯光。中间是 3D 预览舞台，右侧展示关键帧和资产信息。

  • 动作：生成机器人角色节奏动作。
  • 运镜：生成围绕角色的摄像机轨迹。
  • 灯光：生成舞台主光与轮廓光 Cue。
  • 原型角色：使用 REPO.png 作为参考，并提供混元 3D 生成的 FBX 下载。

GitHub Pages 发布

  1. 提交并推送到 EndToFx/StageRehearsal 的 main 分支。
  2. 打开 GitHub 仓库的 Settings -> Pages。
  3. Source 选择 Deploy from a branch。
  4. Branch 选择 main，Folder 选择 / (root)。

Unreal 实时联动

公网 Demo 不能直接访问本机 Unreal。需要在本机运行 WebSocket bridge，然后让页面连接 ws://localhost:9090。

python bridge/ue_bridge.py --port 9090

建议推送 FPS 保持在 8-10，避免编辑器和浏览器过载。

数据导出与第三方导入

点击“导出 JSON”会下载包含 animation、camera、lighting 的结构化文件。Unreal、Blender、TouchDesigner 都可以通过脚本读取该 JSON，并映射到相机、灯光和角色控制器。

{
  "animation": [],
  "camera": [],
  "lighting": []
}
